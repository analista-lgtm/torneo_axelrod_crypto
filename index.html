Torneo evolutivo multi-activo: censo total de estrategias, auditoría anti-overfitting y convergencia universal.

Metodología

Cada activo pasa por la misma tubería estándar de datos (fuente Yahoo Finance, misma ventana temporal, misma limpieza y codificación de estados) para garantizar comparabilidad. El mercado se codifica como memoria binaria de N días (subió = 1, bajó = 0) y se hace competir a todas las estrategias posibles (N=2 → 16, N=3 → 256, N=4 → 65,536) en modo Long/Short sobre el retorno del día siguiente. El Sharpe se anualiza con 365 días para cripto y 252 para mercados tradicionales.
